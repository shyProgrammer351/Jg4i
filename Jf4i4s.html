문제

1.6km까지 기본료가 4800원이고 131m 당 100원이 부과된다고 가정할 때, 1.6km 이상의 이동거리(m)를 입력하면 택시비를 계산하는 문제입니다 실행순서를 클릭하세요 
택시의 이동거리(m)를 입력하세요(1600이상): 1600
1600m 이동시 택시요금은 4800원입니다

프로그램 코드

	import java.util.Scanner;

		

	public class TaxiFare

	{

1		public static void main( String[] args ) {

			Scanner scan = new Scanner( System.in );

			System.out.print( "택시의 이동거리(m)를 입력하세요(1600이상): " );

2			int distance = scan.nextInt();

			

3			int taxiFare = 4800 + ( distance - 1600 + 130 ) / 131 * 100;

		

4			System.out.print( distance + "m 이동시 택시요금은 " + taxiFare + "원입니다 " );

			scan.close();

5		}

	}
